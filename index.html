jQuery Saison 3 : La revanche du Script

Épisode 3 : Construisez une page html avec un bouton et un champ number. Le but est de trouver un nombre entre 0 et 100. À chaque réponse la page répond : plus, moins ou correcte. Quand la réponse est trouvée, on obtient le nombre d'essais que l'on a fait.

À toi de jouer, devine la valeur du nombre mystérieux !
